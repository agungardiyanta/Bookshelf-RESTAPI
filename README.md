# BookshelfAPI
Simple REST API Project Using Hapi framework
1. API can save name of book
Method : POST
URL : /books
2. API show all books
Method : GET
URL: /books
3. API show detail of book
Method : GET
URL: /books/{bookId}
4. API can change detail of book
Method : PUT
URL : /books/{bookId}
5. API delete a book from list
Method : DELETE
URL: /books/{bookId}
